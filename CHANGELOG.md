# Changelog

このプロジェクトの主な変更を記録します。形式は[Keep a Changelog](https://keepachangelog.com/ja/1.1.0/)を参考にし、バージョン番号は[Semantic Versioning](https://semver.org/lang/ja/)に従います。

## [Unreleased]

## [1.0.3] - 2026-07-10

### Added

- 係数入力欄でReturn/Enterキーを押した時に確認ダイアログを表示し、承認後に適用する動作を追加

## [1.0.2] - 2026-06-23

### Changed

- 係数スライダーの範囲を1%から250%に調整

### Fixed

- 係数の数値入力で1%未満または500%超過の値を警告し、対象外の値が適用されないよう修正

## [1.0.1] - 2026-06-22

### Fixed

- グローバルプロセスカラーを実色へ展開してから係数を適用
- CEPパネルの最小サイズ指定を削除

## [1.0.0] - 2026-06-20

### Added

- 選択オブジェクトの色成分に対する係数処理
- RGB、CMYK、Gray、スポットカラーへの対応
- 個別カラーチャンネルの指定
- 塗り、線、テキスト、線形・円形グラデーション停止色への対応
- 未対応オブジェクトとフリーグラデーションを保護するスキップ処理
- 10%、50%、150%、200%のプリセット
