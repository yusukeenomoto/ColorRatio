# Contributing

ColorRatioへの不具合報告、改善提案、プルリクエストを歓迎します。

## Issue

- IllustratorとmacOSのバージョンを記載してください。
- 対象オブジェクトの種類、色空間、操作手順を記載してください。
- 再現用ファイルには個人情報や機密情報を含めないでください。
- データ破損のおそれがある場合は、その旨を明記してください。

## Pull Request

1. 変更範囲を一つの目的に絞ります。
2. Illustratorで対象機能と未対応オブジェクトのスキップ動作を確認します。
3. UIを変更した場合はスクリーンショットを添付します。
4. ユーザー向けの変更は`CHANGELOG.md`の`Unreleased`へ追記します。

署名証明書、パスワード、生成済みZXPはコミットしないでください。
